## Инструкция по получению Google API Key

1. Зарегистрировать (если еще нет) аккаунт на https://www.google.com
2. Перейти на https://aistudio.google.com/ и зарегистрироваться
3. Внизу слева нажать на `Get API key`
4. Вверху справа нажать `Create API key`
5. В открывшемся окне ввести (придумать) название ключа и выбрать свой проект
6. Скопировать ключ и вставить его в файле **.env**
