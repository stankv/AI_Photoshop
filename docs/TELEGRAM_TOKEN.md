## Инструкция по получению TELEGRAM TOKEN

1. В Telegram в поле поиска ввести **BotFather**, перейти в чат
2. В чате нажать кнопку "Запустить" или ввести команду `/start`
3. Выбрать или ввести команду `/newbot`
4. Ввести имя создаваемого бота, например **my_photoshop_bot**
5. Ввести никнейм нового бота, название должно быть уникальным и заканчиваться на **bot**
6. Скопировать токен и вставить его в файле **.env**
7. перейти в чат созданного бота по ссылке из сообщения c токеном
